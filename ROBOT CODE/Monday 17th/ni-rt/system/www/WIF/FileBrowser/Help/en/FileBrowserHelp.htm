Remote File Browser Page

Use this page to manage files on the remote device. This page includes the following components:

  • Upload File—Transfers files from the local computer to the remote device. When you click this button, the browser displays a local file browser. Use the local file browser to select one or more files to transfer to the current directory on the remote device.
  • Download File—Transfers a file from the remote device to the local computer. When you click this button, the browser displays a local file browser. Use the local file browser to select a destination directory where you want to save the file on the local computer, then enter a name for the file.
  • Rename—Edits the name of the selected file. You cannot use this page to edit the name of a directory.
  • Edit File—Opens the selected file in a text editor. Use this feature to edit text files in-place on the remote device.
  • Cut—Removes the selected file(s) and places the file(s) on the clipboard to be pasted. This command does not apply to directories.
  • Copy—Copies the selected file(s) to the clipboard to be pasted. This command does not apply to directories.
  • Paste—Pastes the contents of the clipboard to the current remote directory.
  • Create Directory—Creates a new directory under the current remote directory. When you click this button, the browser displays a text entry dialog where you can name the new directory.
  • Create File—Creates a new empty file in the current remote directory. The name of the new file begins with NewFile followed by an integer suffix.
  • Delete—Deletes the selected files and/or directories.
  • File Name—Displays the name of each file and directory contained in the current remote directory.
  • File Size (bytes)—Displays the size (in bytes) of each file contained in the current remote directory.
  • Modification Time—Displays the time and date of the last update of each file contained in the current remote directory.
