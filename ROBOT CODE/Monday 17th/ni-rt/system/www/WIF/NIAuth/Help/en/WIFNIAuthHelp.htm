Security Configuration Page

Use this page to complete the following tasks:

  • Secure a target
  • Change a user's password
    Note  By default, the admin password is blank. To secure the Web Application Server, National Instruments recommends changing the password for this user as soon as possible.
  • Add or remove users to or from the list of users that can configure the Web Application Server on this target
  • Assign users to a group or remove users from a group
  • Grant or deny permissions to users and groups

Securing a Target

National Instruments recommends that you complete the following steps to secure a target:

  1. Change the administrator password. By default, this password is blank.
    Note  Changing the admin password also changes the password you use to access the target from Measurement & Automation Explorer, the LabVIEW Real-Time Module, and other applications.
    • Click the Users tab on the left side of this page and select the admin user.
    • Click the Change Password button on the right side of this page.
    • Enter and confirm the new password and click OK.
  2. Remove permissions from the everyone group. By default, this group has all permissions.
    • Click the Groups tab on the left side of this page and select the everyone group.
    • Select permissions from the Permissions list on the right side of this page and click the - to remove them.
    • Apply the changes to the target.
  3. Click Save to save changes.

Changing a User's Password

Complete the following steps to change a user's password:

  1. Click the Users tab on the left side of this page and select the user's name.
  2. Click the Change Password button on the right side of this page.
  3. Enter and confirm the new password and click OK.
  4. Click Save to save changes.

Adding and Removing Users

Complete the following steps to add a user:

  1. Click the Users tab on the left side of this page.
  2. Click the + button at the bottom of this tab.
  3. Enter a User Name on the right side of this dialog box.
  4. Change the user's password.
  5. Grant or deny permissions to the user.
  6. (Optional) Assign the user to a group. New users are automatically added to the everyone group.
  7. Click Save to save changes.

Complete the following steps to remove a user:

  1. Select the user from the Users tab on the left side of this page.
  2. Click the - button at the bottom of this tab.
  3. Click Save to save changes.

Assigning Users to a Group or Removing Users from a Group

Complete the following steps to assign users to a group:

  1. Select the group from the Groups tab on the left side of this page.
  2. Locate the This group contains the following users field on the right side of this page.
  3. Click the + button at the bottom of this field.
  4. Select the user(s) to add to the group and click OK.
  5. Click Save to save changes.

Complete the following steps to remove users from a group:

  1. Select the group from the Groups tab on the left side of this page.
  2. Locate the This group contains the following users field on the right side of this page.
  3. Select the user you want to remove from the group.
  4. Click the - button at the bottom of this field.
  5. Click Save to save changes.

Granting and Denying Permissions to a User or Group

Complete the following steps to grant or deny permissions to a user or group:

  1. Select the user or group from the Users or Groups tab on the left side of this page.
  2. Locate the This user/group has the following permissions field on the right side of this page.
  3. Use the + and - buttons at the bottom of this field to grant or deny permissions.
  4. Click Save to save changes.

List of Permissions

Desktop Computers

  • GetDB—View users, groups, and permissions on the Security Configuration page.
  • GetSystemConfiguration—View the settings on the System Configuration, System Settings, Network Configuration, and Time Configuration pages.
  • GetWSAPIKey—Get the Web services API key.
  • ManageExtensions—Enable and disable Web configuration tools using the Installed Configuration Tools page.
  • ManageWS—Pause, resume, and restart published Web services on the Web Services Management page.
  • NIWebCer—Replicate the hidden files, such as SSL certificates and signing requests.
  • SetDB—Modify the user's own password and comments on the Security Configuration page. A user must have both the GetDB and SetDB permissions to modify their own permissions and other user, group, and permission data.
  • SetSystemConfiguration—Modify the settings on the System Settings page.
  • SetWSAPIKey—Set the Web services API key.
  • UndeployWS—Unpublish published Web services on the Web Services Management page.
  • WIFConfigureAppServer—Configure the System and Application Web Servers on the Web Server Configuration page. Refer to the LabVIEW Help for information about the System and Application Web Servers.

Remote Targets

  • fsdelete—Delete files on the Remote File Browser page.
  • fsread—Read files on the Remote File Browser page.
  • fswrite—Edit files on the Remote File Browser page.
  • GetDB—View users, groups, and permissions on the Security Configuration page.
  • GetSystemConfiguration—View the settings on the System Configuration, System Settings, Network Configuration, and Time Configuration pages.
  • manageextensions—Enable and disable Web configuration tools using the Installed Configuration Tools page.
  • ManageWS—Pause, resume, and restart published Web services on the Web Services Management page.
  • reboot—Reboot the remote system.
  • remoteshell—Reserved for future use.
  • SetDB—Modify the user's own password and comments on the Security Configuration page. A user must have both the GetDB and SetDB permissions to modify their own permissions and other user, group, and permission data.
  • setlocale—Modify the Locale on the System Settings page.
  • setrtlockpassword—Set the password to lock the remote system.
  • setsystemconfiguration—Modify the General Settings on the System Settings page.
  • UndeployWS—Unpublish published Web services on the Web Services Management page.
  • viewconsoleoutput—View output on the Console page.
  • WIFConfigureAppServer—Configure the System and Application Web Servers on the Web Server Configuration page. Refer to the LabVIEW Help for information about the System and Application Web Servers.
