Software Management Page

Use this page to manage software on the target. This page includes the following components:

  • Currently Installed Software—Displays the software currently installed on the target. Click an item name to view the software version and description.

    You can expand this list box by clicking and dragging the right edge of the box.
  • Refresh—Updates the Currently Installed Software page and the list of repositories.
Note  You cannot update or install software from the NI Web-based Configuration & Monitoring Interface. You must launch Measurement & Automation Explorer (MAX) to update or install software on the target.
