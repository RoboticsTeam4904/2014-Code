Installed Configuration Tools

Use this page to enable and disable the Web Monitoring and Configuration pages installed on the remote device. This page includes the following components.

  • Control Name—Displays the name of each Web Monitoring and Configuration page installed on the remote device.
  • Is Enabled?—Specifies whether to enable each Web Monitoring and Configuration page installed on the remote device.
  • Description—Describes the functionality of each Web Monitoring and Configuration page installed on the remote device.
